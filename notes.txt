How to use the application.

1. Insert new item to shopping list
	Method type: POST
	Url: http://localhost:8080/shopping
	Input json:
	   {
    	"name" : "itemOne",
    	"quantity" : "1"
		}
		
2. Get all items in the shopping list
	Method type: GET
	Url: http://localhost:8080/shopping
	Returns: List of all items

3. Get items by id	
	Method type: GET
	Url: http://localhost:8080/shopping/1
	Returns: Item of that id
	{
    	"id": 1,
    	"name": "itemOne",
    	"quantity": 1
	}
4. Updated Item
	Method type: PUT
	Url: http://localhost:8080/shopping/1
	Input json:
	{
        "id": 1,
        "name": "itemupdated",
        "quantity": 5
	}	

5. Delete item
	Method type: DELETE
	Url: http://localhost:8080/shopping/1
